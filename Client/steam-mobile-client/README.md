This is the Flutter client for **QUp!**, a cross-platform Steam companion app that helps users find multiplayer games shared between friends.  
This client handles the UI, Steam login, game browsing, matching features, and communication with the Node.js API.

---

## Requirements

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.x or later)
- Android Studio, Xcode, or VS Code (for device emulation or a physical device)
- A valid Steam account (for authentication)
- The QUp! API server running locally or hosted (see main project README)
